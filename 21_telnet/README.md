## 21_telnet - создание текста в ASCI art

Это приложение с помощью сервиса telehack.com создает ASCI art изображение с 
указанным в качестве аргумента командной строки текстом. 

### Сборка 
```bash
cmake .
cmake --build .
```
### Использование

Приложение принимает аргументы командной строки:
1. -f <название шрифта> - не обязательный параметр, по умолчанию "standard"
2. Все оставшиеся аргументы - текст, который нужно преобразовать в ASCI art

```bash
% ./21_telnet otus.ru
figlet /standard otus.ru
          _
   ___   | |_   _   _   ___       _ __   _   _
  / _ \  | __| | | | | / __|     | '__| | | | |
 | (_) | | |_  | |_| | \__ \  _  | |    | |_| |
  \___/   \__|  \__,_| |___/ (_) |_|     \__,_|

% ./21_telnet -f bell otus.ru
figlet /bell otus.ru
         .
   __.  _/_   ,   .   ____   .___  ,   .
 .'   \  |    |   |  (       /   \ |   |
 |    |  |    |   |  `--.    |   ' |   |
  `._.'  \__/ `._/| \___.' / /     `._/|

```
