## 16_crc32 - расчет CRC32 для больших файлов

Рассчитывает CRC32 сумму идя по большому файлу и последовательно отображая его фрагмент в память с помощью mmap. 
Для каждого фрагмента рассчитывается crc на основе данных фрагмента и предыдущей crc.

### Сборка
```bash
cmake .
cmake --build .
```

### Запуск
```bash
./16_crc32 file.bin
deadbeef
```
